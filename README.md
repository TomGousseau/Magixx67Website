## Getting Started

```bash
# Install dependencies
npm install

# Run with Turbopack
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) and become 67 sigma.

## Adding Your Images

Replace the `PlaceholderImage` components with actual Next.js Image components:

```tsx
import Image from 'next/image'

<Image 
  src="/your-image.png" 
  alt="Description"
  width={800}
  height={600}
/>
```
